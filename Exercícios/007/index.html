Princípais formatações


Negrito / Destaque

Nessa frase temos um termo em negrito usando as tags <b> e </b> (não semântica).

Nesta frase, tenho um termo em destaque usando as tags <strong> e </strong> (semântica).


Itálico / Ênfase

Nesta frase, temos um termo em itálico usando as tags <i> e </i> (não semântica).

Nesta frase, temos um termo em ênfase usando a tag <em> e </em> (semântica).


Texto Marcado

Podemos criar também texto marcado usando a tags <mark> e </mark>.

E no outro outro parágrafo temos um texto marcado no final.


Texto Grande e Pequeno

Estamos criando texto grande e um texto pequeno.


Texto Deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado usando as tags <del> e </del>.


Texto Inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois. Tags <ins> e </ins>(semântica).

Existe também o sublinhado com as tags <u> e </u> (não semântica).

Para inserir coisas de tipo x20+3.
